### 第四步: 应用Ken Burns效果 (30分钟) ⭐核心步骤

#### 4.1 打开Inspector面板

1. 确保在 **Edit** 页面
2. 点击右上角 **Inspector** 按钮
3. Inspector面板出现在右上角
4. 如果看不到，按 Cmd+4

#### 4.2 为第一张图片创建Ken Burns效果

**详细操作**:

1. 在时间线上选择 **shot_001**
2. 在 Inspector 面板中:

   **A. 启用缩放动画**
   - 找到 **Zoom** 部分
   - 点击 Zoom 前面的 **菱形图标** (启用关键帧)

   **B. 设置开始关键帧 (0秒)**
   - 确保播放头在0秒位置
   - 设置 Zoom 值: **1.000** (100%)
   - 设置 Position X: **0**
   - 设置 Position Y: **0**

   **C. 移动到结束位置**
   - 在时间线上，将播放头拖到 **6秒** 位置
   - 或点击当前时间码，输入: 00:00:06:00

   **D. 设置结束关键帧 (6秒)**
   - 设置 Zoom 值: **1.150** (115%)
   - 设置 Position X: **50** (向右移动50像素)
   - 设置 Position Y: **30** (向下移动30像素)

   **E. 预览效果**
   - 按空格键播放
   - 应该看到图片缓慢放大+轻微向右下移动

**为什么这样设置**:
- Zoom 1.000→1.150: 缓慢放大15%，温和不突兀
- Position偏移50/30: 轻微平移，增加动感
- 6秒时长: 观众有足够时间看清内容

#### 4.3 让动画更平滑 (可选但推荐)

1. 在Inspector下方，找到关键帧编辑器
2. 右键点击关键帧点
3. 选择 **Interpolation** (插值)
4. 选择 **Ease In Out** (缓入缓出)
5. 这样运动会更自然，不生硬

#### 4.4 批量应用到所有图片

**方法: 复制粘贴属性**

1. 选择 shot_001 (已设置好效果)
2. 按 **Cmd+C** 复制
3. 选择其他15张图片:
   - 点击 shot_002
   - 按住 Shift
   - 点击 shot_016
   - 全选
4. 右键 → **Paste Attributes...** (粘贴属性)
5. 在弹出窗口中:
   - ✅ 勾选 **Transform** (变换)
   - ✅ 勾选 **Keyframes** (关键帧)
   - ❌ 取消勾选其他选项
6. 点击 **OK**

**验证**:
- 所有图片应该都有动画关键帧
- 每张图片下方的轨道上显示菱形关键帧图标

**高级: 为不同图片设置不同方向** (可选)

如果想要每张图片都独特，可以手动调整:

```
镜头1,5,9,13:  Position X = 50,  Y = 30   (右下)
镜头2,6,10,14: Position X = -50, Y = 30   (左下)
镜头3,7,11,15: Position X = 50,  Y = -30  (右上)
镜头4,8,12,16: Position X = -50, Y = -30  (左上)
```

这样可以创造视觉节奏感。

### 第五步: 添加转场效果 (5分钟)

1. **选择转场位置**:
   - 在时间线上，点击两个片段之间的连接线
   - 会高亮显示

2. **添加转场**:
   - 按 **Cmd+T** (Mac)
   - 或右键 → "Add Transition" → "Cross Dissolve"

3. **调整转场时长**:
   - 双击转场图标 (中间的过渡图标)
   - 在 Inspector 中
   - Duration: **1秒** (00:00:24 frames)

4. **批量添加**:
   - 对所有15个转场点重复上述操作
   - 或 Shift+点击多个转场点
   - 按 Cmd+T 批量添加

**推荐转场**:
- **Cross Dissolve** (交叉溶解) - 最常用
- **Fade** (淡入淡出) - 柔和
- **Dip to Color** (颜色浸入) - 创意

## 💡 专业技巧

### 技巧1: 变化动画速度

不同类型的镜头可以使用不同的缩放比例:

```
标题卡 (1-4):     Zoom 100%→110% (温和)
场景镜头 (5-10):   Zoom 100%→115% (明显)
图解 (11-13):     Zoom 100%→110% (适中)
要点 (14-16):     Zoom 100%→120% (突出)
```

### 技巧2: 调整关键帧曲线

让动画更自然:
1. 在关键帧编辑器中
2. 右键关键帧
3. "Interpolation" → "Ease In Out"
4. 或 "Bezier" 手动调整曲线手柄

### 技巧3: 使用调整图层

如果想要批量修改:
1. 创建 "Adjustment Layer"
2. 对调整图层应用Ken Burns
3. 所有片段继承效果

## 🆘 常见问题

### Q: Ken Burns效果在哪里?

**A**:
- Inspector 面板 → Zoom 部分
- 点击 Zoom 前面的**菱形图标**启用动画
- 没有这个图标，动画不会启用

### Q: 关键帧怎么添加?

**A**:
- 移动播放头到起始位置
- 设置参数 (Zoom, Position)
- 自动创建第一个关键帧
- 移动播放头到结束位置
- 修改参数
- 自动创建第二个关键帧

### Q: 动画太生硬怎么办?

**A**:
- 右键关键帧
- "Interpolation" → "Ease In Out"
- 或 "Bezier" 手动调整曲线

### Q: 转场添加失败?

**A**:
- 确保相邻片段有足够的时长
- 每个片段至少比转场时长长1秒
- 我们的设置: 6秒片段 + 1秒转场 = 满足要求

### Q: 导出的视频很大?

**A**:
- 在 Deliver 页面
- 降低 "Limit Data Rate" 到 10 Mbps
- 或使用 "H.265" 编码 (文件更小但兼容性稍差)

### Q: 音视频不同步?

**A**:
- 检查音频片段位置
- 确保音频开头与视频开头对齐
- 必要时使用 "Trim" 工具调整
