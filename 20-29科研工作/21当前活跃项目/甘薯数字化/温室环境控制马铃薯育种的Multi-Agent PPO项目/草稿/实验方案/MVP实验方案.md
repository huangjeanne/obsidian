# 温室环境控制马铃薯育种Multi-Agent PPO项目 - MVP实验方案

## 🎯 **最小可行产品(MVP)实验方案**

**核心理念**: 花最少的钱，验证最关键的技术
**总预算**: 19,100元
**实验周期**: 4个月
**验证目标**: Multi-Agent PPO在农业应用的技术可行性

---

## 🏠 **最小化实验设置**

### **实验规模**
- **温室数量**: 2个小型温室 (每个10m²)
- **对照设计**: 1个Multi-Agent PPO + 1个传统控制
- **实验周期**: 3个月 (马铃薯完整生长周期)
- **植株数量**: 每个温室20株马铃薯

### **硬件配置 (最小化)**
```
每个温室配置：
├── 传感器 (必需)
│   ├── 温湿度传感器 x1 (200元)
│   ├── 土壤湿度传感器 x2 (300元)
│   └── 光照传感器 x1 (150元)
├── 执行器 (简化)
│   ├── 小型加热器 x1 (300元)
│   ├── 小型风扇 x1 (200元)
│   ├── 滴灌系统 x1 (400元)
│   └── LED补光灯 x1 (500元)
├── 控制器
│   └── 树莓派4B x1 (400元)
└── 其他材料 (500元)

单个温室成本: 3,050元
两个温室总成本: 6,100元
```

---

## 🤖 **简化的Multi-Agent设计**

### **2个核心智能体** (而非4个)
1. **环境控制智能体**: 管理温度、湿度、通风
2. **水肥管理智能体**: 管理灌溉和营养供给

### **简化的状态空间**
```python
# 环境控制智能体状态
env_state = [temp, humidity, light, hour_of_day, days_planted]

# 水肥管理智能体状态
water_state = [soil_moisture, plant_stage, days_since_water, weather]
```

### **简化的动作空间**
```python
# 环境控制智能体动作 (3个连续动作)
env_actions = [heating_power, fan_speed, led_intensity]

# 水肥管理智能体动作 (2个连续动作)
water_actions = [irrigation_duration, nutrient_concentration]
```

## 📈 **MVP评估指标**

### **核心指标** (必须测量)
1. **植株生长速度**: 每周测量株高、叶片数
2. **最终产量**: 收获时称重对比
3. **资源消耗**: 电费、水费记录
4. **控制精度**: 环境参数偏差统计

### **简化数据采集**
- **频率**: 每小时记录一次传感器数据
- **人工测量**: 每周一次植株生长数据
- **成本记录**: 每日资源消耗记录
