## 🔧 PAI 使用规则

### 当 PAI 生成内容时，应：

1. **识别内容类型**
   - 视频方案 → `63多媒体素材/视频创作方案/`
   - 文案脚本 → `62文章与脚本/`
   - Skills 文档 → `工具与配置/Skills文档/`
   - 多媒体素材 → `63多媒体素材/`（对应子目录）

2. **创建主题文件夹**
   - 使用有意义的文件夹名称
   - 例如：`tomato-marketing-video`、`spring-campaign-2026`

3. **保持组织结构**
   - 相关内容放在同一主题文件夹
   - 大型项目单独创建文件夹
   - 小型内容可以合并到同类文件夹

4. **文件命名规范**
   - 使用清晰的文件名
   - 例如：`README.md`、`SUMMARY.md`、`quick-start.sh`
   - 避免使用 `temp`、`test` 等模糊名称
