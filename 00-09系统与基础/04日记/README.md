## 🎯 使用规范

### 1. 文件命名规范

**日常日记**：
```
YYYY-MM-DD.md
示例：2026-02-28.md
```

**主题日记**：
```
YYYY-MM-DD-主题.md
示例：2026-02-28-项目复盘.md
```

**月度复盘**：
```
YYYY-MM-复盘.md
示例：2026-02-复盘.md
```

**会议记录**：
```
YYYY-MM-DD-会议名称.md
示例：2026-02-28-项目讨论会.md
```

### 2. 目录组织

**按年分组**：
```
04日记/
├── 2026/
│   ├── 01/
│   ├── 02/
│   └── ...
├── 2027/
│   ├── 01/
│   └── ...
```

**原则**：
- ✅ 每月一个文件夹
- ✅ 跨年自动创建新年份文件夹
- ✅ 使用数字编码保持排序稳定

## 📝 模板使用

### 日常日记模板

**位置**：`模板/日常日记模板.md`

**使用方法**：
1. 复制模板到当前月份
2. 重命名为当天日期
3. 填写内容

**快速创建**：
```bash
# 在 Obsidian 中
1. 按 Cmd/Ctrl + P 打开命令面板
2. 输入 "创建日记"
3. 选择模板
4. 自动生成当天日记
```

### 月度复盘模板

**位置**：`模板/月度复盘模板.md`

**使用时机**：
- 每月最后一天
- 或次月前3天
- 完成上月总结

**包含内容**：
- 目标达成情况
- 重点成就
- 关键洞察
- 下月计划

### 会议记录模板

**位置**：`模板/会议记录模板.md`

**适用场景**：
- 项目讨论会
- 学术会议
- 导师学生见面
- 合作伙伴会议

### 第一步：创建本月目录

```bash

mkdir -p 2026/02
```

### 第二步：创建今日日记

```bash
# 方法1：复制模板
cp 模板/日常日记模板.md 2026/02/2026-02-28.md
```

- 🌅 早晨计划

- 💭 今日反思

- ⏰ **固定时间**：每天早晨或晚上

- 📈 **数据分析**：统计本月成就
- 🚀 **下月规划**：设定新的目标

建议设置：
- `Cmd/Ctrl + Shift + D` - 创建今日日记
- `Cmd/Ctrl + Shift + R` - 创建月度复盘
- `Cmd/Ctrl + Shift + M` - 创建会议记录

## ❓ 常见问题

### Q1: 如何处理跨月的内容？
**A**: 按开始日期存放，在日记中用链接关联相关日期

### Q2: 是否需要每天写日记？
**A**: 不强制。建议至少每周写一次，重要事件随时记录

### Q3: 如何回顾旧日记？
**A**:
- 方法1：通过全局索引（01索引）
- 方法2：使用 Dataview 查询
- 方法3：直接浏览对应文件夹

### Q4: 可以在手机上写日记吗？
**A**: 可以。使用 Obsidian 移动端，配合 iCloud 同步
