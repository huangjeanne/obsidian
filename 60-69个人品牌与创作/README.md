## 🚀 使用 PAI 生成内容

PAI 会自动将生成的内容保存到对应的文件夹中：

### 视频创作
```
"帮我做一个产品的营销视频"
→ 保存到：63多媒体素材/视频创作方案/[项目名称]/
```

### 文案创作
```
"写一个营销文案"
→ 保存到：62文章与脚本/营销文案/[主题].md
```

### 媒体下载
```
"下载 5 张星空图片"
→ 保存到：63多媒体素材/图片素材/[主题]/
```

### 文档生成
```
"生成配图"
→ 保存到：63多媒体素材/图片素材/[文档名称]-images/
```
