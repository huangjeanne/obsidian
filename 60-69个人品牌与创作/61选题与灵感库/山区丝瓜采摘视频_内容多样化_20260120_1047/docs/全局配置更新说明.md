## 🆕 新增功能

### 1. 对话脚本生成器

**位置**: `scripts/generate_dialogue_script.py`

**功能**:
- 为科普视频生成教师-学生对话脚本
- 一问一答式，适合科普教学
- 每个镜头匹配图片内容

**使用方法**:
```bash
python scripts/generate_dialogue_script.py
```

**输出**:
- `dialogue_script_teacher_student.json`
- `dialogue_script_teacher_student.txt`

---

### 2. 双人TTS生成器

**位置**: `scripts/generate_dialogue_audio.py`

**功能**:
- 批量生成对话音频
- 教师音色: 男声（原野，沉稳专业）
- 学生音色: 女声（晓曼，年轻活泼）
- 为每个镜头生成匹配的音频

**使用方法**:
```bash
python scripts/generate_dialogue_audio.py
```

**输出**:
- `dialogue_audio/shot_001_teacher.mp3`
- `dialogue_audio/shot_001_student.mp3`
- ... (32个音频文件)

---

### 3. 多音频视频合成器

**功能**:
- 为每个镜头添加对应的对话音频
- 自动对齐音视频
- 合成最终视频

**输出**:
- `丝瓜采摘机器人_双人对话版.mp4`

---

### 4. 图片文字检查器

**位置**: `scripts/check_chinese_text.py`

**功能**:
- 检查提示词中的中文比例
- 验证图片使用中文
- 生成检查报告

**使用方法**:
```bash
python scripts/check_chinese_text.py
```
