# 温室环境控制马铃薯育种Multi-Agent PPO实验验证方案

## 1. 实验假设

### 1.1 主假设
**H₁**: Multi-Agent PPO在马铃薯收获预测准确性上显著优于传统控制方法
- **零假设(H₀)**: Multi-Agent PPO与传统方法在收获预测准确性上无显著差异
- **备择假设(H₁)**: Multi-Agent PPO的预测准确性显著高于传统方法(p < 0.05)

### 1.2 子假设
**H₂**: 多智能体协作比单智能体控制更有效
**H₃**: Multi-Agent PPO在能耗效率上优于传统方法
**H₄**: 系统在不同马铃薯品种上具有良好的泛化性能

## 2. 实验设计

### 2.1 对照组设置

| 组别 | 控制方法 | 描述 | 样本量 |
|------|----------|------|--------|
| **对照组1** | 传统PID控制 | 基于专家经验的PID参数设置 | n=12个温室区域 |
| **对照组2** | 单智能体PPO | 单一智能体控制所有环境参数 | n=12个温室区域 |
| **对照组3** | 人工专家控制 | 农业专家手动调节环境参数 | n=6个温室区域 |
| **实验组** | Multi-Agent PPO | 4个智能体协作控制 | n=12个温室区域 |

### 2.2 实验环境层次

#### 第一层：仿真环境验证
- **环境**: 基于DSSAT植物生长模型的仿真器
- **时长**: 3个月
- **目的**: 算法基础验证和参数调优

#### 第二层：小规模温室试验
- **环境**: 4个独立温室区域(每个50m²)
- **时长**: 6个月(1个完整生长周期)
- **目的**: 真实环境初步验证

#### 第三层：大规模温室验证
- **环境**: 42个温室区域(每个100m²)
- **时长**: 12个月(2个完整生长周期)
- **目的**: 系统性能和稳定性验证

## 3. 评估指标体系

### 3.1 主要指标

#### 收获预测准确性
```python
# 预测准确性指标
RMSE_yield = sqrt(mean((predicted_yield - actual_yield)²))
MAE_yield = mean(|predicted_yield - actual_yield|)
MAPE_yield = mean(|predicted_yield - actual_yield| / actual_yield) × 100%

# 收获时间预测
RMSE_time = sqrt(mean((predicted_harvest_time - actual_harvest_time)²))
MAE_time = mean(|predicted_harvest_time - actual_harvest_time|)
```

#### 实际产量对比
- **单位面积产量** (kg/m²)
- **马铃薯品质等级** (A级、B级、C级比例)
- **平均单薯重量** (g)

### 3.2 次要指标

#### 能耗效率
```python
# 能耗指标
Energy_efficiency = Total_yield / Total_energy_consumption  # kg/kWh
Energy_cost = Total_energy_consumption × Energy_price      # 元/m²
```

#### 环境控制稳定性
- **温度控制精度**: |T_actual - T_target| < 1°C 的时间比例
- **湿度控制精度**: |H_actual - H_target| < 5% 的时间比例
- **响应时间**: 环境参数调节到目标值的平均时间

```
温室区域传感器布局 (10m × 10m):
```

### 4.2 数据采集频率
- **环境参数**: 每分钟采集一次
- **植株生长数据**: 每周人工测量一次
- **图像数据**: 每天拍摄一次植株照片
- **产量数据**: 收获时精确称重

### 4.3 数据质量控制
- **传感器校准**: 每月校准一次
- **数据异常检测**: 基于3σ原则自动标记异常值
- **缺失数据处理**: 使用线性插值或LSTM预测填补

## 6. 统计分析方案

### 6.1 样本量计算
基于预期效应量(Cohen's d = 0.8)和统计功效(1-β = 0.8):
```python
# 使用G*Power计算
α = 0.05  # 显著性水平
β = 0.2   # II类错误概率
effect_size = 0.8  # 预期效应量

# 每组最小样本量
n_per_group = 16  # 考虑20%脱落率，实际需要20个样本
```

### 6.2 统计检验方法
- **主要分析**: 单因素方差分析(One-way ANOVA)
- **事后检验**: Tukey HSD多重比较
- **非参数检验**: Kruskal-Wallis检验(数据不满足正态性时)
- **置信区间**: 95%置信区间

### 6.3 多重比较校正
使用Bonferroni校正控制家族错误率:
```python
α_corrected = α / number_of_comparisons
# 对于4组比较，α_corrected = 0.05 / 6 = 0.0083
```

## 10. 可能问题及解决方案

### 10.1 技术问题
**问题1**: 多智能体训练不稳定
- **解决方案**: 采用渐进式训练，先单智能体后多智能体
- **备选方案**: 使用MADDPG算法作为备选

**问题2**: 实际环境与仿真差异大
- **解决方案**: 域适应技术，真实数据微调
- **备选方案**: 增加仿真环境的复杂度

### 10.2 实验问题
**问题3**: 样本量不足
- **解决方案**: 延长实验周期，增加温室数量
- **备选方案**: 使用Bootstrap重采样增加样本

**问题4**: 环境控制失效
- **解决方案**: 人工干预机制，安全边界设置
- **备选方案**: 切换到传统PID控制
