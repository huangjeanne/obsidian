# 农作物种质资源鉴定技术规程 甘薯

# 1 范围

本标准规定了甘薯[Ipomoea batatas(L.)Lam.]种质资源鉴定的技术要求和方法。

本标准适用于甘薯[Ipomoea batatas(L.)Lam.]栽培种种质资源的植物学特征、生物学特性、品质性状和抗性的鉴定，甘薯属的其他种参考本鉴定技术规程。

# 3 技术要求

# 3.1 鉴定地点

环境条件应能够满足甘薯植株正常生长及其性状的正常表达。

# 3.2 样本采集

应在植株正常生长情况下采集样本。鉴定应连续进行2个～3个生长周期。特征特性的鉴定应不少于10株，产量和品质鉴定应同时种植对照品种。

# 4 鉴定方法

# 4.1 植物学特征

# 4.1.1 株型

栽插后  $30\mathrm{d}\sim 40\mathrm{d}$  ，观察整个小区植株生长状况，根据茎蔓和分枝的形态与空间分布确定株型。株型分为直立(主茎和分枝短，茎蔓无伏地)、半直立(主茎和部分分枝较短，主蔓  $< 1 / 3$  伏地生长)、匍匐（主茎和部分分枝长，主蔓  $\geqslant 1 / 3$  伏地生长)、攀缘(茎蔓和部分分枝纤细，茎端有缠绕圈)。

# 4.1.2 顶叶色

栽插后  $40\mathrm{d}\sim 50\mathrm{d}$  ，观察主茎第一片展开叶的颜色。分为金黄、浅绿、绿、褐绿、紫绿、褐、红、浅紫、紫。

# 4.1.22 薯皮色

收获时，观察薯块表皮的颜色。以最大相似原则分为白、淡黄、黄、褐、橘红、粉红、红、紫、深紫。

# 4.1.23 皮层颜色

收获时，刮开外表皮确定皮层的颜色。分为白、淡黄、黄、褐、橘红、粉红、红、紫、深紫。

# 4.1.24 薯皮厚度

收获时，随机测量10个薯块横切面皮层的厚度，结果以平均值表示，精确到  $0.1\mathrm{mm}$ 。

# 4.1.25 薯肉主色

# A.1 范围

# A.2 对照品种

中度感病品种：冲绳100号。

# A.3 鉴定方法

# A.3.1 病菌分离与保存

采集病害组织样本接种于PDA培养基上，从菌落中分离和纯化病菌后置于冰箱保存。当年分离的病菌需当年用。

将保存的菌种用无菌水稀释，接种在健康的薯块切片上，置于  $26\%$  条件下保湿培养，待薯片长满黑色的孢子囊壳和孢子囊孢子后，用无菌水冲洗，配成浓孢子悬浮液备用。

# A.3.3 接种方法

每个供试品种选取3块中等大小、表面光滑、无伤烂的薯块，仔细洗净，晾干，编号备用。一个薯块为一个重复。将浓孢子悬浮液稀释为  $1\times 10^{7}$  个孢子/mL(或显微镜160倍下每视野40个～60个孢子)的悬浮液备用。用医用注射针头蘸取配好的菌液，穿刺测试的薯块接种；每个薯块接种15个点，深度 $0.5\mathrm{cm}$  。接种后的薯块放入消毒的塑料箱中，加盖消毒纱布，置于  $25\% \sim 28\%$  条件下的恒温室中，每天上午、下午用温水浇湿覆盖纱布进行保湿。培养  $10\mathrm{d}$  后，切开薯块测量病斑直径和深度，计算平均病斑直径和深度，与对照品种(冲绳100号)比较确定抗性级别。

$$

# 甘薯染色体倍性鉴定

# J.1 范围

本附录适用于甘薯种质资源全部材料的倍性鉴定。

# J.2 仪器设备

普通光学显微镜

# J.3 试剂和溶液配置

0.002%的8-羟基喹啉  $\left(\mathrm{C}_{9} \mathrm{H}_{7} \mathrm{NO}\right)$  、尿素  $\left(\mathrm{CO}\left(\mathrm{NH}_{2}\right)_{2}\right)$  、盐酸(HCl)、冰乙酸  $\left(\mathrm{CH}_{3} \mathrm{COOH}\right)$  、碱性品红  $\left(\mathrm{C}_{20} \mathrm{H}_{20} \mathrm{~N}_{3} \mathrm{Cl} \cdot 4 \mathrm{H}_{2} \mathrm{O}\right)$  、苯酚  $\left(\mathrm{C}_{6} \mathrm{H}_{5} \mathrm{OH}\right)$  、甲醛  $\left(\mathrm{HCHO}\right)$  、乙醇  $\left(\mathrm{CH}_{3} \mathrm{CH}_{2} \mathrm{OH}\right)$  和山梨醇  $\left(\mathrm{C}_{6} \mathrm{H}_{14} \mathrm{O}_{5}\right)$  。

采用卡宝品红染液进行染色，其配制方法如下：

1）原液A：称取  $3\mathrm{g}$  碱性品红溶于  $100\mathrm{mL}70\%$  乙醇中(可永久保存)；  
2）原液B：取  $10\mathrm{mL}$  原液A加入  $90\mathrm{mL}5\%$  的苯酚水溶液，在  $37^{\circ}\mathrm{C}$  条件下温育  $2\mathrm{h}\sim 4\mathrm{h}$  （在2周内使用）；  
3）原液C：取  $55\mathrm{mL}$  原液B加入  $6\mathrm{mL}$  冰乙酸和  $6\mathrm{mL}$  甲醛，充分混匀；  
4）卡宝品红染液：取  $10\mathrm{mL}\sim 20\mathrm{mL}$  原液C加入约  $80~\mathrm{mL}~45\%$  的冰乙酸和  $1\mathrm{g}$  山梨醇(可永久保存)。

# J.4 分析步骤

# J.4.1 取样

选择生长旺盛的植株，将剪取部分幼嫩根，放在清水中，加适量尿素，经常换水，待根尖长到  $1\mathrm{cm}$  时（此时根尖分生能力强，可获得较多的分裂相），选择生长粗壮的根尖剪下。

# J.4.2 待测试样制备

用  $0.002\%$  的8-羟基喹啉处理根尖  $2\mathrm{h}\sim 3\mathrm{h}$  （温度保持在  $25^{\circ}C$  以下）。

# J.4.3 待测试样的固定、保存和解离

将预处理后的根尖用水冲洗3次～5次后，用卡诺固定液(乙醇：冰乙酸  $= 3:1$  )固定  $4\mathrm{h}$  以上，可将固定好的材料转入  $70\%$  的乙醇中，在  $4^{\circ}\mathrm{C}$  的条件下可保存数月。

将固定好的根尖在  $1\mathrm{mol} / \mathrm{L}$  盐酸， $60^{\circ}\mathrm{C}$  条件下恒温处理  $5\mathrm{min} \sim 10\mathrm{min}$ ，然后用水冲洗3次～5次，将洗净盐酸后的材料置于  $45\%$  的冰乙酸水溶液中。

# J.4.4 待测试样的涂片、染色与镜检

选择涂片操作中理想的根尖, 滴上约 1/3 滴卡宝品红染液, 盖上盖玻片, 并用左手食指按住盖片左下角防止盖片滑动, 用镊子头部垂直轻击盖片, 使根尖材料分散; 然后使用滤纸盖住盖片, 用拇指垂直施与压力, 使细胞内的染色体尽量在同一平面上, 以便于染色体数目的统计, 经染色处理后染色体很快即能着色。

在普通的光学显微镜上进行染色体数目的检测即倍性鉴定，记录该根尖细胞的染色体数目，根据每

个鉴定材料的20个受检细胞的镜检结果取平均值，根据表J.1确定该材料的倍性。
