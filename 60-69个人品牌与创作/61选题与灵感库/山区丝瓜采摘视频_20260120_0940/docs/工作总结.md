## ✅ 问题与解决

### 问题1: ImageMagick未安装
**现象**: 部分动画镜头生成失败（10/16）
**原因**: moviepy依赖ImageMagick处理文字
**解决**: `brew install imagemagick`
**结果**: 成功率从37.5%提升到100%

### 问题2: Gemini图像生成失败
**现象**: `gemini-2.0-flash-exp`返回文本而非图像
**原因**: 使用了错误的模型
**解决**: 切换到`nano-banana-pro-preview`
**结果**: 图像生成完全恢复

### 问题3: 视频时长偏短
**现象**: 目标60秒，实际37.92秒（-37%）
**原因**: 中文语速比预期快
**状态**: 待优化（添加语速系数）

### 问题4: 使用单一封面图
**现象**: 所有16个镜头使用同一张图
**原因**: 图像生成之前失败
**状态**: 待优化（每个镜头生成独特图像）
