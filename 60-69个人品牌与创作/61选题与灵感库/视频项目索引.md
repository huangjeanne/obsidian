## 📝 使用说明

### 快速创建新项目

1. **使用skill生成视频**:
```bash
cd ~/.claude/skills/video-generator
python3 scripts/video_workflow.py \
  --topic "你的主题" \
  --style academic \
  --duration 60 \
  --compose-video
```

2. **整理到项目库**:
```bash
# 创建主题文件夹
mkdir "/Users/jeanne/60-69个人品牌与创作/61选题与灵感库/[主题名称]-[日期]"

# 移动生成的文件
cp -r output/* "/Users/jeanne/60-69个人品牌与创作/61选题与灵感库/[主题名称]-[日期]/"
```

3. **更新本索引**:
在上方"项目列表"中添加新项目信息
