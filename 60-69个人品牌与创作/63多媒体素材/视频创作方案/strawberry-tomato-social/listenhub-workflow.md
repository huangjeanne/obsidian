# 使用ListenHub重新生成配音指南

## 🔄 重启Claude Code

ListenHub MCP服务器需要在**新会话**中才能加载。

### 重启方法：

**macOS终端：**
```bash
# 按Ctrl+D退出当前会话，然后重新运行
claude
```

**或者在VS Code中：**
1. 关闭当前Claude Code窗口
2. 重新打开项目
3. 开始新对话

## 🎙️ 重启后的操作

### 步骤1：验证ListenHub已加载

在重启后的新对话中，对我说：
```
验证ListenHub MCP服务器是否已加载
```

我应该能看到可用的ListenHub工具。

### 步骤2：查看中文音色

```
显示ListenHub所有可用的中文音色
```

我会使用 `get_speakers` 工具查询，显示：
- 音色ID
- 音色名称
- 性别
- 试听音频链接

### 步骤3：选择音色

根据您的喜好选择音色（例如：年轻甜美女声）

### 步骤4：重新生成4个场景的配音

我会使用 `create_flowspeech` 工具为每个场景生成新的配音：

**场景1：**
```
哇！这就是传说中的草莓西红柿吗？
```

**场景2：**
```
看看这果形，饱满又可爱～
颜色鲜艳得让人心动！
咬一口汁水丰富，
完全自然成熟～
```

**场景3：**
```
最棒的是～
有机认证，健康安全！
口感评分高达 9.2 分！
甜度刚刚好，12 度甜度指数～
```

**场景4：**
```
还等什么？
立即抢购，
品尝来自大自然的甜蜜吧！
```

### 步骤5：下载并替换音频文件

生成的音频会自动下载，我会帮您：
1. 替换 `public/audio/voice/scene1.mp3` - `scene4.mp3`
2. 保持文件名不变
3. 刷新Remotion Studio预览

### 步骤6：对比音质

在浏览器中预览，对比：
- 🎙️ Edge TTS版（当前版本）
- 🎙️ ListenHub版（新版本）

选择您更喜欢的一个。

## 💡 提示

- ListenHub生成可能需要几分钟（每个场景）
- 生成过程中可以继续其他工作
- 如果ListenHub音质提升不明显，可以保留Edge TTS版本
- 最终导出时选择您最喜欢的版本

## 🎬 导出最终视频

确认配音版本后，我可以帮您：
1. 渲染完整视频（MP4格式）
2. 调整视频参数（分辨率、比特率）
3. 添加字幕（可选）
4. 优化文件大小
