## 📂 各文件夹详细说明

### 1️⃣ 📚 文献库

用于管理和组织研究文献。

#### 文献库/已读论文/

**用途：** 存放已读完并做了笔记的论文

**文件命名建议：**
```
[年份]_[第一作者姓氏]_[核心关键词].md
[年份]_[第一作者姓氏]_[核心关键词].pdf

示例：
2024_Zhang_MARL_Greenhouse_Control.md
2024_Zhang_MARL_Greenhouse_Control.pdf
```

**Markdown 模板：**
```markdown
# 论文标题

**作者：**
**发表年份：**
**期刊/会议：**
**DOI：**

## 📝 研究问题

（论文要解决的核心问题）

## 🔬 方法

（研究方法和技术路线）

## 📊 主要发现

（关键发现和结果）

## 💡 与本研究的关联

（对本研究的启发和借鉴）

## 🎯 可引用的观点

- 观点 1
- 观点 2
```

#### 文献库/待读论文/

**用途：** 存放待阅读的论文

**建议：** 使用文献管理软件（如 Zotero、EndNote）配合此文件夹

### 2️⃣ 📊 实验数据

用于管理实验数据和分析结果。

#### 实验数据/原始数据/

**用途：** 存放传感器采集的原始数据、日志文件等

**文件格式：** .csv, .json, .log, .h5, .pkl 等

**注意事项：**
- ✅ 原始数据不要修改，保持完整性
- ✅ 建议添加 README 说明数据格式
- ✅ 定期备份到云盘或实验室服务器

#### 实验数据/处理后数据/

**用途：** 存放经过清洗、预处理的数据

**处理步骤记录：**
建议创建 `data_processing_log.md` 记录：
- 处理日期
- 处理方法
- 参数选择
- 结果验证

#### 实验数据/结果分析/

**用途：** 存放实验结果图表、分析报告

**文件类型：**
- 📊 图表：.png, .svg, .pdf
- 📄 报告：.md, .docx, .pdf
- 📈 数据：.csv (汇总统计)

## 🎯 使用建议

### 日常使用流程

1. **文献调研**
   - 下载论文 → `文献库/待读论文/`
   - 阅读完 → 移至 `文献库/已读论文/`
   - 写笔记 → 更新 `研究上下文.md`

2. **实验工作**
   - 采集数据 → `实验数据/原始数据/`
   - 处理数据 → `实验数据/处理后数据/`
   - 分析结果 → `实验数据/结果分析/`

3. **论文撰写**
   - 写草稿 → `草稿/论文草稿/`
   - 做图表 → `代码/分析脚本/`
   - 生成配图 → `实验数据/结果分析/`

4. **团队协作**
   - 记录会议 → `草稿/会议记录/`
   - 更新进度 → `科研团队文档/研究上下文.md`

### 版本管理建议

**使用 Git 管理的项目：**
- ✅ 代码文件夹
- ✅ 草稿文件夹（Markdown 格式）
- ✅ 文献库（Markdown 笔记）

**不纳入 Git 的内容：**
- ❌ 实验数据（太大）
- ❌ PDF 论文（可单独管理）
- ❌ 模型权重文件（使用 Git LFS 或云盘）
