## ✅ 问题修复历史

### 问题1: 图像生成返回文本
**现象**: `gemini-2.0-flash-exp`返回文本而非图像
**原因**: 使用了错误的模型
**解决**: 切换到`nano-banana-pro-preview`
**状态**: ✅ 已修复

### 问题2: ImageMagick未安装
**现象**: 部分动画镜头生成失败（10/16）
**原因**: moviepy依赖ImageMagick处理文字
**解决**: `brew install imagemagick`
**结果**: 成功率从37.5%提升到100%
**状态**: ✅ 已修复

### 问题3: 所有镜头使用同一张图
**现象**: 16个镜头全部使用相同的cover.png
**原因**: 图像生成失败后使用fallback
**解决**: 批量生成16张独特图片
**验证**: MD5哈希值全部不同
**状态**: ✅ 已修复
