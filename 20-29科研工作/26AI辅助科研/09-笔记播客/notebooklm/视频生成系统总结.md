# 视频生成脚本系统 - 总结文档

## 项目概述

本项目实现了一个基于Gemini 3 Pro的视频脚本生成和分镜头解析系统，专门用于知识科普、教学视频和学术论文讲解。

---

## 已完成模块

### 1. 脚本格式规范 (`视频生成脚本格式规范.md`)

**功能：** 定义了完整的Markdown脚本格式规范

**核心内容：**
- YAML frontmatter元数据格式
- 场景/分镜头格式定义
- 11种视觉类型规范（title_card, text_bullet, formula, chart, diagram, scene, code, quote, comparison, timeline, process）
- 动态背景规范
- 5种预设风格模板（academic, tech, minimal, handdrawn, 3d）
- 情感和转场效果定义

**特点：**
- 支持多角色对话
- 情感控制（neutral, enthusiastic, engaged, curious, serious, friendly, calm, excited）
- 背景类型（static, dynamic, video, color）
- 转场效果（none, fade, slide, zoom, wipe, dissolve）

### 2. 脚本生成器 (`scripts/script_generator.py`)

**功能：** 使用Gemini 3 Pro API从主题生成完整视频脚本

**核心类：** `ScriptGenerator`

**主要方法：**
```python
# 生成完整脚本
generate_script(topic, style, duration, aspect_ratio, voices, additional_context)

# 精修单个场景
refine_scene(script, scene_number, refinement_instruction)

# 分析脚本统计
analyze_script(script)
```

**使用示例：**
```bash
# 生成学术风格脚本
python3 scripts/script_generator.py \
  --topic "什么是相对论" \
  --style academic \
  --duration 180 \
  --voices teacher student \
  --output script.md

# 精修场景3
python3 scripts/script_generator.py \
  --refine script.md \
  --scene 3 \
  --instruction "让对话更生动，增加互动"

# 分析脚本
python3 scripts/script_generator.py \
  --analyze script.md
```

**特点：**
- 支持5种预设风格
- 自动生成6-8个场景
- 对话自然流畅
- 自动计算时长
- 支持中英文冒号

### 3. 场景解析器 (`scripts/scene_parser.py`)

**功能：** 将Markdown脚本解析为结构化的场景数据

**核心类：** `ScriptParser`

**数据结构：**
```python
@dataclass
class SceneMetadata:
    scene_number: int
    title: str
    role: str
    emotion: str
    duration: int
    background_type: str
    transition: str
    transition_duration: float

@dataclass
class VisualContent:
    type: str
    title: str
    content: str
    animation: str
    layout: str
    extra: Dict[str, Any]

@dataclass
class Dialogue:
    speaker: str
    text: str
    pause: float

@dataclass
class Scene:
    metadata: SceneMetadata
    dialogues: List[Dialogue]
    visual: Optional[VisualContent]
    dynamic_background: str
    raw_text: str
```

**使用示例：**
```bash
# 解析并验证脚本
python3 scripts/scene_parser.py script.md --validate

# 导出JSON和场景列表
python3 scripts/scene_parser.py script.md \
  --output-json script.json \
  --output-list scenes_list.md
```

**特点：**
- 完整的场景解析
- 支持中英文冒号
- 脚本验证功能
- JSON和Markdown导出
- 统计信息生成

### 4. 视觉映射器 (`scripts/visual_mapper.py`)

**功能：** 将视觉类型映射到具体的生成指令（Manim、Nano Banana）

**核心类：** `VisualMapper`

**支持的视觉类型：**

| 类型 | Manim | Nano Banana | 说明 |
|------|-------|-------------|------|
| title_card | ✓ | ✓ | 标题卡片 |
| text_bullet | ✓ | ✓ | 文字列表 |
| formula | ✓ | ✓ | 数学公式 |
| chart | ✓ | ✓ | 图表 |
| diagram | ✓ | ✓ | 示意图 |
| code | ✓ | | 代码展示 |
| quote | | ✓ | 引用 |
| comparison | | ✓ | 对比表格 |
| scene | | ✓ | 场景描述 |

**使用示例：**
```python
mapper = VisualMapper(style="tech")

instruction = mapper.map_visual(
    visual_type="formula",
    title="质能方程",
    content="E = mc²",
    animation="fade_in",
    layout="center"
)

print(instruction.manim_code)          # Manim代码
print(instruction.nanobanana_prompt)   # Nano Banana提示词
```

**特点：**
- 自动选择生成引擎
- LaTeX公式转换
- 风格模板配置
- 背景提示词生成

**特点：**

### 6. Listenhub FlowSpeech 客户端 (`scripts/listenhub_client.py`)

**功能：** Listenhub FlowSpeech API 客户端，支持异步 TTS 语音合成

**核心类：** `ListenhubClient`

**主要方法：**
```python
# 创建 FlowSpeech 任务
create_flowspeech(text, speaker_id, speed, emotion)

# 获取 FlowSpeech 任务状态
get_flowspeech_status(flowspeech_id)

# 等待 FlowSpeech 任务完成
wait_for_flowspeech(flowspeech_id, poll_interval, max_poll_time)

# 文本转语音（包含完整的异步工作流）
text_to_speech(text, voice_id, output_path, speed, emotion, wait)

# 获取可用说话人列表
get_speakers()

# 批量文本转语音
batch_text_to_speech(texts, voice_id, output_dir, speed, emotion)

# 向后兼容方法（已弃用）
clone_voice_from_file(audio_file, voice_name, description)
clone_voice_from_url(audio_url, voice_name, description)
```

**使用示例：**
```bash
# 列出可用说话人
python3 scripts/listenhub_client.py --action list

# 生成语音（使用说话人ID）
python3 scripts/listenhub_client.py --action tts \
  --text "你好，世界" --speaker-id speaker_001 --output hello.mp3

# 不等待完成（异步模式）
python3 scripts/listenhub_client.py --action tts \
  --text "长文本" --speaker-id speaker_001 --no-wait
```

**特点：**
- 基于 FlowSpeech 异步 API（创建 → 轮询 → 下载）
- 自动API密钥加载
- 支持情感和语速控制
- 批量TTS处理
- 完整错误处理
- 预定义说话人列表（通过 `get_speakers()`）

**注意：**
- FlowSpeech 是异步 API，默认会等待任务完成
- 使用 `--no-wait` 可以立即返回 `flowspeech_id` 进行后台处理
- 传统的 "声音克隆"（从音频文件创建自定义语音）可能不被支持
- API 端点：`/v1/flowspeech`（创建）、`/v1/flowspeech/{id}`（状态）、`/v1/speakers`（列表）

### 8. 完整工作流 (`scripts/video_workflow.py`)

**功能：** 整合所有组件的完整视频生成工作流

**核心类：** `VideoGenerationWorkflow`

**工作流步骤：**
1. 生成脚本（从主题）或加载现有脚本
2. 解析场景
3. 生成视频封面
4. 为每个场景生成音频（TTS）
5. 为每个场景生成视觉内容（封面/动画）
6. （可选）合成最终视频

**使用示例：**
```bash
# 从主题生成完整视频
python3 scripts/video_workflow.py --topic "什么是相对论" --style academic

# 使用现有脚本
python3 scripts/video_workflow.py --script script.md --generate-visuals

# 只生成音频
python3 scripts/video_workflow.py --script script.md --generate-audio --no-visuals

# 指定输出目录
python3 scripts/video_workflow.py --topic "AI教程" --output ./my_video
```

**特点：**
- 一键式完整工作流
- 模块化组件初始化
- 灵活的选项控制
- 自动目录管理
- 进度显示
