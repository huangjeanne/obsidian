## 🔧 技术实现

### FFmpeg滤镜链

```python
filter_complex = (
    "scale=1376:768:flags=bicubic,"      # 高质量缩放
    "fade=t=in:st=0:d=0.8:alpha=1,"      # 淡入0.8秒
    "fade=t=out:st=5.2:d=0.8:alpha=1,"  # 淡出0.8秒
    "fps=24"                             # 24帧/秒
)
```

**效果说明**:
- **scale**: 标准化到1376x768
- **fade in**: 前面0.8秒淡入（alpha通道）
- **fade out**: 最后0.8秒淡出（总6秒-0.8秒=5.2秒开始）
- **fps**: 统一帧率

### 音频合成

```bash
ffmpeg -i video.mp4 -i audio.mp3 \
  -c:v copy \          # 复制视频流
  -c:a aac \           # AAC音频编码
  -shortest \          # 使用最短的流
  output.mp4
```

## 🎨 动画效果详解

### 淡入淡出时间轴

```
镜头1 (6秒):
├─────── 淡入 0.8秒 ─────┐
│                       │
│    [内容展示 4.4秒]   │
│                       │
└───────── 淡出 0.8秒 ─────┘
```

**视觉效果**:
- 0.0-0.8秒: 从黑色渐入到图片
- 0.8-5.2秒: 完整显示图片
- 5.2-6.0秒: 图片渐出到黑色

### 转场效果

**镜头间过渡**:
```
镜头1 (5.2秒结束) → [黑色0.8秒] → 镜头2 (0.8秒淡入)
```

实际上：
- 镜头1的最后0.8秒淡出到黑色
- 镜头2的前0.8秒从黑色淡入
- 视觉上形成自然的过渡
