# CNKI高级检索技能 - 使用示例

## 📚 支持的来源类型

技能现已支持所有知网来源类别，可以根据你的需求灵活选择：

| 来源类别 | 关键词 | 适用场景 |
|---------|-------|---------|
| **CSSCI（默认）** | CSSCI、C刊、南大核心 | 社科类高质量期刊 |
| **SCI** | SCI、Science Citation Index | 国际权威科学期刊 |
| **北大核心** | 北大核心、中文核心 | 国内重要期刊 |
| **EI** | EI、Engineering Index | 工程技术类期刊 |
| **CSCD** | CSCD、中国科学引文数据库 | 自然科学核心期刊 |
| **CMSCI** | CMSCI、科技核心 | 科技类期刊 |

## 🎯 使用示例

### 示例1：检索CSSCI论文（默认）

```
帮我在知网检索"数字化转型"相关论文
```

**AI行为**：
- 默认来源类别 = CSSCI
- 解析关键词 = 数字化转型
- 确认后执行检索

### 示例2：明确指定CSSCI

```
用知网高级检索搜索"人工智能"主题的CSSCI论文
```

**AI行为**：
- 来源类别 = CSSCI
- 关键词 = 人工智能
- 勾选CSSCI来源后检索

### 示例3：检索SCI论文

```
帮我在知网检索"机器学习"的SCI论文
```

**AI行为**：
- 来源类别 = SCI
- 关键词 = 机器学习
- 取消"全部期刊"，勾选"SCI"

### 示例4：检索北大核心期刊

```
检索"企业绩效"相关的北大核心论文
```

**AI行为**：
- 来源类别 = 北大核心
- 关键词 = 企业绩效
- 勾选"北大核心"选项

### 示例5：检索EI论文

```
用知网高级检索搜索"深度学习"的EI期刊论文
```

**AI行为**：
- 来源类别 = EI
- 关键词 = 深度学习
- 勾选"EI"选项

### 示例6：多关键词检索

```
检索"数字化转型与企业绩效"的CSSCI论文
```

**AI行为**：
- 来源类别 = CSSCI
- 第一组关键词 = 数字化转型 + 数字化变革
- 第二组关键词 = 企业绩效 + 企业业绩 + 组织绩效
- 两组关键词用OR连接

### 示例7：指定多种来源

```
检索"人工智能"相关的CSSCI和北大核心论文
```

**AI行为**：
- 来源类别 = CSSCI + 北大核心（多选）
- 关键词 = 人工智能
- 同时勾选"CSSCI"和"北大核心"

## 💡 技巧提示

### 1. 省略来源类型（使用默认CSSCI）

如果你主要检索社科类论文，直接说关键词即可，AI会默认使用CSSCI：

```
检索"供应链管理"相关论文
```

### 2. 明确指定来源类型

对于理工科或其他领域，明确说明来源类型：

```
检索"图像识别"的SCI论文
检索"材料科学"的EI论文
检索"生态学"的CSCD论文
```

### 3. 同义词自动扩展

AI会自动为关键词添加同义词，你也可以明确指定：

```
检索"机器学习 + 深度学习 + 神经网络"的论文
```

### 4. 多关键词组合

使用"与"、"和"等词连接多个主题，AI会自动设置OR关系：

```
检索"绿色创新与企业绩效"的CSSCI论文
```

## 📊 检索结果

无论选择哪种来源类型，检索流程都相同：

1. ✅ 打开知网高级检索页面
2. ✅ 选择"学术期刊"类别
3. ✅ 勾选你指定的来源类别
4. ✅ 输入关键词（含同义词扩展）
5. ✅ 执行检索
6. ✅ 按被引量排序
7. ✅ 切换50条/页
8. ✅ 打开摘要视图
9. ✅ 提取前100篇论文信息
10. ✅ 导出为Excel文件

## 📁 输出文件

文件名格式：`知网检索结果_{关键词}_{来源类型}_{日期}.xlsx`

示例：
- `知网检索结果_人工智能_CSSCI_20260211.xlsx`
- `知网检索结果_机器学习_SCI_20260211.xlsx`
- `知网检索结果_企业绩效_北大核心_20260211.xlsx`

## ❓ 常见问题

**Q: 我不知道该选哪个来源类型怎么办？**

A: 默认CSSCI即可，AI会在检索前向你确认来源类别。你也可以咨询导师或参考所在学科的期刊目录。

**Q: 可以同时检索多个来源吗？**

A: 可以！说"检索CSSCI和北大核心论文"，AI会同时勾选多个来源。

**Q: 某个来源类型没有结果怎么办？**

A: AI会告知你实际结果数量。如果结果为0，建议尝试其他来源类型或调整关键词。

**Q: 检索速度会很慢吗？**

A: AI会自动控制操作间隔（1-2秒）避免触发反爬，完整检索约需2-5分钟。
