## ⚠️ 重要说明

### 自动检索限制

由于知网的反爬机制和安全验证，**完全自动化的检索目前需要人工配合**：

1. **验证码处理**：需要手动完成滑块验证
2. **页面结构变化**：知网页面可能动态加载元素
3. **批量操作限制**：知网限制批量导出功能

### 推荐工作流程

**方案A：半自动模式（推荐）**

1. 使用技能文档指导检索步骤
2. 手动完成验证和关键操作
3. 手动导出检索结果
4. 使用提供的Excel模板整理

**方案B：参考文档模式**

1. 阅读详细的检索指南
2. 按步骤手动执行检索
3. 导出并整理结果
