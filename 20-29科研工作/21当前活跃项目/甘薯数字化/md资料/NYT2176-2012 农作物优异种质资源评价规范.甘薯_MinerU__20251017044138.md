# （规范性附录）

# 甘薯茎线虫病(Ditylenchus destructor Thorne)抗性鉴定

# A.1 范围

本附录适用于甘薯茎线虫病(Ditylenchus destructor Thorne)抗性的鉴定。

# A.2 鉴定步骤

# A.2.1 接种体制备

收获时，从病地采集带虫薯块，利用贝尔曼漏斗法或浅盘法分离茎线虫，将线虫置于无菌水中， $4^{\circ}\mathrm{C}$  保存备用。

# A.2.2 接种体繁殖

于3月上旬用孔径  $5\mathrm{mm}$  灭菌打孔器在甘薯薯块中部打孔，深度达薯块中心，用灭菌的注射器向孔内滴注500条茎线虫，封口置于  $25^{\circ}\mathrm{C}$  下并保湿，繁殖茎线虫，至栽种时切块，每块约为  $10\mathrm{g}$ 。

# A.2.3 接种与调查

在专用病圃地，每穴施1个接种体甘薯切块，然后栽春薯苗。垄距  $0.8\mathrm{m}$ ，每个材料栽5株，株距 $0.2\mathrm{m}$ ，每隔10个材料设置高感品种栗子香作为对照，每个材料3次重复。植株生长至  $140\mathrm{d} \sim 150\mathrm{d}$  收获薯块，并在每个薯块的上部和中部各横切1刀，观察记录茎线虫病的危害情况，按照表A.1标准确定病级。

表 A. 1 甘薯茎线虫病害分级标准  

<table><tr><td>病害分级</td><td>薯块症状</td></tr><tr><td>0级</td><td>无病</td></tr><tr><td>1级</td><td>为害面积占薯块横切面积25%以下</td></tr><tr><td>2级</td><td>为害面积占薯块横切面积25%~50%</td></tr><tr><td>3级</td><td>为害面积占薯块横切面积50%~75%</td></tr><tr><td>4级</td><td>为害面积占薯块横切面积75%以上</td></tr></table>

# A.3 结果计算

依据表A.1病害级别，按式(A.1)计算病情指数  $(DI)$

$$
D I = \frac {\sum \left(X _ {i} N _ {j}\right)}{4 n} \times 1 0 0 \tag {A.1}
$$

式中：

$DI$  ——病情指数，单位为百分率  $(\%)$ ；

$X_{i}$  ——薯块发病级数，单位为级；

$N_{j}$  ——各级感病薯数，单位为个；

$n$  ——调查总薯块数，单位为个；

4 最高病级数。

计算结果表示到小数点后1位。

# 附录B

# （规范性附录）

# 甘薯茎叶菜用食味鉴定与产量测定

# B.1 范围

本附录适用于甘薯茎叶菜用食味鉴定与产量测定。

# B.2 鉴定步骤

# B.2.1 种植

采用畦作方式，每小区种植行数不少于6行，行距不大于  $30~\mathrm{cm}$  ，株距不大于  $20~\mathrm{cm},$  3次重复，栽插后按常规栽培管理。同时设福薯7-6为对照。

注：提供的对照种质福薯7-6是为了方便本标准的使用，并不代表对该种质的认可和推荐，任何可以得到与该对照种质相同结果的种质均可作为对照种质。

# B.2.2 采摘

在适宜种植(4月～9月)的季节内，每隔  $10\mathrm{d}$  采摘一次，连续采摘6次。采摘茎尖  $10\mathrm{cm}$  范围内的茎和叶片，每次称茎叶产量，计算累计小区产量，结果重复3次，以平均值表示。

# B.2.3 茎尖处理

称取采摘的茎尖约  $150\mathrm{g}$ ，洗净，并用  $1000\mathrm{mL}100^{\circ}\mathrm{C}$  沸水烫至  $2\mathrm{min}$ 。

# B.3 评价

根据茎尖茸毛的有无、烫后颜色、香味、苦涩味、甜味、滑腻感等感官指标综合评价食味，参照对照品种福薯7-6，按照表B.1的规定确定其食味。

# B.4 精密度

甘薯茎叶菜用食味鉴定均应以即采即评价的方式进行，每次评价应至少有3位经过培训的技术人员同时现场独立评价计分。最终评价结果以各次采摘综合评价的平均值确定。
