# 视频生成脚本系统 - 总结文档

## 项目概述

本项目实现了一个基于Gemini 3 Pro的视频脚本生成和分镜头解析系统，专门用于知识科普、教学视频和学术论文讲解。

---

## 已完成模块

### 1. 脚本格式规范 (`视频生成脚本格式规范.md`)

**功能：** 定义了完整的Markdown脚本格式规范

**核心内容：**
- YAML frontmatter元数据格式
- 场景/分镜头格式定义
- 11种视觉类型规范（title_card, text_bullet, formula, chart, diagram, scene, code, quote, comparison, timeline, process）
- 动态背景规范
- 5种预设风格模板（academic, tech, minimal, handdrawn, 3d）
- 情感和转场效果定义

**特点：**
- 支持多角色对话
- 情感控制（neutral, enthusiastic, engaged, curious, serious, friendly, calm, excited）
- 背景类型（static, dynamic, video, color）
- 转场效果（none, fade, slide, zoom, wipe, dissolve）

---

### 2. 脚本生成器 (`scripts/script_generator.py`)

**功能：** 使用Gemini 3 Pro API从主题生成完整视频脚本

**核心类：** `ScriptGenerator`

**主要方法：**
```python
# 生成完整脚本
generate_script(topic, style, duration, aspect_ratio, voices, additional_context)

# 精修单个场景
refine_scene(script, scene_number, refinement_instruction)

# 分析脚本统计
analyze_script(script)
```

**使用示例：**
```bash
# 生成学术风格脚本
python3 scripts/script_generator.py \
  --topic "什么是相对论" \
  --style academic \
  --duration 180 \
  --voices teacher student \
  --output script.md

# 精修场景3
python3 scripts/script_generator.py \
  --refine script.md \
  --scene 3 \
  --instruction "让对话更生动，增加互动"

# 分析脚本
python3 scripts/script_generator.py \
  --analyze script.md
```

**特点：**
- 支持5种预设风格
- 自动生成6-8个场景
- 对话自然流畅
- 自动计算时长
- 支持中英文冒号

---

### 3. 场景解析器 (`scripts/scene_parser.py`)

**功能：** 将Markdown脚本解析为结构化的场景数据

**核心类：** `ScriptParser`

**数据结构：**
```python
@dataclass
class SceneMetadata:
    scene_number: int
    title: str
    role: str
    emotion: str
    duration: int
    background_type: str
    transition: str
    transition_duration: float

@dataclass
class VisualContent:
    type: str
    title: str
    content: str
    animation: str
    layout: str
    extra: Dict[str, Any]

@dataclass
class Dialogue:
    speaker: str
    text: str
    pause: float

@dataclass
class Scene:
    metadata: SceneMetadata
    dialogues: List[Dialogue]
    visual: Optional[VisualContent]
    dynamic_background: str
    raw_text: str
```

**使用示例：**
```bash
# 解析并验证脚本
python3 scripts/scene_parser.py script.md --validate

# 导出JSON和场景列表
python3 scripts/scene_parser.py script.md \
  --output-json script.json \
  --output-list scenes_list.md
```

**特点：**
- 完整的场景解析
- 支持中英文冒号
- 脚本验证功能
- JSON和Markdown导出
- 统计信息生成

### 4. 视觉映射器 (`scripts/visual_mapper.py`)

**功能：** 将视觉类型映射到具体的生成指令（Manim、Nano Banana）

**核心类：** `VisualMapper`

**支持的视觉类型：**

| 类型 | Manim | Nano Banana | 说明 |
|------|-------|-------------|------|
| title_card | ✓ | ✓ | 标题卡片 |
| text_bullet | ✓ | ✓ | 文字列表 |
| formula | ✓ | ✓ | 数学公式 |
| chart | ✓ | ✓ | 图表 |
| diagram | ✓ | ✓ | 示意图 |
| code | ✓ | | 代码展示 |
| quote | | ✓ | 引用 |
| comparison | | ✓ | 对比表格 |
| scene | | ✓ | 场景描述 |

**使用示例：**
```python
mapper = VisualMapper(style="tech")

instruction = mapper.map_visual(
    visual_type="formula",
    title="质能方程",
    content="E = mc²",
    animation="fade_in",
    layout="center"
)

print(instruction.manim_code)          # Manim代码
print(instruction.nanobanana_prompt)   # Nano Banana提示词
```

**特点：**
- 自动选择生成引擎
- LaTeX公式转换
- 风格模板配置
- 背景提示词生成


**特点：**

### 8. FFmpeg 视频合成器 (`scripts/video_composer.py`)

**功能：** 使用 FFmpeg 将音频和视觉组件合成为最终视频

**核心类：** `VideoComposer`

**主要方法：**
```python
# 合成单个场景（音频 + 图片/视频 → 视频片段）
compose_scene(audio_path, visual_path, output_path, duration, aspect_ratio, transition)

# 合成完整视频（多场景片段 → 最终视频）
compose_full_video(segments, output_path, quality, aspect_ratio, background_music)

# 添加背景音乐
_add_background_music(video_path, music_path, output_path, music_volume)

# 合并多个片段（带转场效果）
_merge_segments(segments, output_path, quality)
```

**使用示例：**
```bash
# 合成单个场景
python3 scripts/video_composer.py compose-scene \
  --audio scene1.mp3 \
  --visual scene1.png \
  --output scene1.mp4 \
  --duration 30 \
  --aspect-ratio 16:9

# 合成完整视频（使用配置文件）
python3 scripts/video_composer.py compose-full \
  --config scenes_config.json \
  --output final_video.mp4 \
  --quality high

# 检查 FFmpeg
python3 scripts/video_composer.py --check-ffmpeg
```

**支持的转场效果：**
- `none` - 无转场（直接拼接）
- `fade` - 淡入淡出
- `slide` - 滑动切换
- `zoom` - 缩放切换
- `wipe` - 擦除切换
- `dissolve` - 溶解效果

**视频质量预设：**
- `low` - 低质量（CRF 28，文件小）
- `medium` - 中等质量（CRF 23）
- `high` - 高质量（CRF 18，推荐）
- `ultra` - 超高质量（CRF 15，文件大）

**特点：**
- 支持图片和视频作为视觉素材
- 自动淡入淡出效果
- 转场效果支持
- 背景音乐混合
- 可配置视频质量
- 临时文件自动清理

### 9. 完整工作流 (`scripts/video_workflow.py`)

**功能：** 整合所有组件的完整视频生成工作流

**核心类：** `VideoGenerationWorkflow`

**工作流步骤：**
1. 生成脚本（从主题）或加载现有脚本
2. 解析场景
3. 生成视频封面
4. 为每个场景生成音频（TTS）
5. 为每个场景生成视觉内容（封面/动画）
6. （可选）合成最终视频（FFmpeg）
7. （可选）添加背景音乐

**使用示例：**
```bash
# 从主题生成完整视频
python3 scripts/video_workflow.py --topic "什么是相对论" --style academic

# 使用现有脚本
python3 scripts/video_workflow.py --script script.md --generate-visuals

# 只生成音频
python3 scripts/video_workflow.py --script script.md --generate-audio --no-visuals

# 生成并合成完整视频
python3 scripts/video_workflow.py --topic "AI教程" --compose-video --video-quality high

# 指定输出目录
python3 scripts/video_workflow.py --topic "AI教程" --output ./my_video
```

**特点：**
- 一键式完整工作流
- 模块化组件初始化
- 灵活的选项控制
- 自动目录管理
- 进度显示
- **FFmpeg 视频合成**（新增）

## 进度条显示 ✅ (新增)

系统集成了 `tqdm` 进度条库，为长时间运行的任务提供实时进度反馈。

### 支持进度条的操作

| 操作 | 进度条描述 | 示例 |
|------|-----------|------|
| TTS 音频生成 | 显示每条对话的生成进度 | `生成音频 [teacher]: 45%|████▌     | 5/11 [00:15<00:19,  3.21s/个]` |
| FlowSpeech 等待 | 显示任务状态轮询 | `等待 TTS 完成 (状态: processing): 50%|████▌     | 15/30 [00:30<00:30,  2.00s/次]` |
| 视觉内容生成 | 显示场景渲染进度 | `生成场景 3 (Manim): 50%|████▌     | 3/6 [01:00<01:00, 20.00s/个]` |
| 视频合成 | 显示场景片段合成 | `合成场景 3/6: 50%|████▌     | 3/6 [00:10<00:10,  3.33s/个]` |

### 特性

- **自动降级**：如果未安装 tqdm，系统自动降级为文本进度显示
- **状态显示**：实时显示当前操作状态（pending/processing/completed）
- **时间估算**：显示已用时间和预计剩余时间
- **操作详情**：显示当前处理的对象（speaker、场景编号等）

### 安装

```bash
pip install tqdm
```

### 配置

所有进度条默认启用，可通过参数关闭：

```python
# 关闭特定操作的进度条
result = client.text_to_speech(text, voice_id, show_progress=False)
result = composer.compose_full_video(segments, output_path, show_progress=False)
```
