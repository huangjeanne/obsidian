## 🎵 完整音频配置

### 语音文件命名
- `voiceover.mp3` - 主解说（完整）
- `scene1.mp3` - 场景 1 配音
- `scene2.mp3` - 场景 2 配音
- `scene3.mp3` - 场景 3配音
- `scene4.mp3` - 场景 4配音

### 音效文件命名
- `sfx_attention.wav` - 吸引注意音（0.0s）
- `sfx_spring.mp3` - 弹簧音（0.5s）
- `sfx_pop_1.mp3` - 特点出现音1（4.0s）
- `sfx_pop_2.mp3` - 特点出现音2（4.4s）
- `sfx_pop_3.mp3` - 特点出现音3（4.8s）
- `sfx_pop_4.mp3` - 特点出现音4（5.2s）
- `sfx_bgm.mp3` - 背景音乐（4-20s）
- `sfx_card_appear.mp3` - 卡片出现（9.0s）
- `sfx_number_roll.mp3` - 数字滚动（10.5s）
- `sfx_progress.mp3` - 进度条（12.0s）
- `sfx_button_pop.mp3` - 按钮弹出（16.0s）
- `sfx_end.mp3` - 收尾音（19.0s）

### 背景音乐
- **风格**：轻快、活泼、甜蜜
- **乐器**：尤克里里、钢琴、轻微的鼓点
- **节奏**：120 BPM
- **音量**：-20dB（不覆盖人声）

## 🎬 集成到 Remotion

### 代码示例
```tsx
import { Audio } from "remotion";
import { staticFile } from "remotion/static-file";

export const MyVideo = () => {
  return (
    <AbsoluteFill>
      {/* 背景音乐 */}
      <Audio
        src={staticFile("bgm.mp3")}
        volume={0.3}
      />

      {/* 场景配音 */}
      <Audio
        src={staticFile("voiceover.mp3")}
        startFrom={0}
        endAt={240} // 8秒
      />
    </AbsoluteFill>
  );
};
```
