## 二、配图需求清单

### 配图1：封面图

**位置**：文章开头，标题下方

**功能**：吸引注意力，概括主题

**尺寸**：900 x 500 px（公众号首图标准）

**设计要求**：
- **主视觉**：多个小机器人/智能体协作的场景
- **构图**：左侧单一Agent → 右侧多个Agent协作，用箭头连接
- **色彩**：科技蓝背景，橙色高亮"协作"概念
- **文字**：主标题"AI Agent的下一个战场"（可选，视排版需要）

**参考描述**：
```
[左侧] 一个孤独的小机器人
   ↓ 渐变箭头（橙色）
[右侧] 三个小机器人围成一圈协作
   背景：科技感的网格/节点连接
```

**工具建议**：Midjourney / DALL-E / 专业设计师

---

### 配图2：5大热点信息图

**位置**：数据说话章节

**功能**：直观展示5大热点和讨论量

**尺寸**：900 x 600 px

**设计要求**：
- **形式**：横向柱状图或卡片式布局
- **内容**：
  ```
  1. AI Agent自主决策能力  ████ 1,200条
  2. 记忆存储和检索方案  ███  980条
  3. 多Agent协作      ██   856条
  4. 信任与安全       ██   742条
  5. 工具调用能力      ██   695条
  ```
- **视觉**：每个热点配一个简化的图标
  - 自主决策：🧠 大脑/思考图标
  - 记忆存储：💾 数据库/记忆图标
  - 多Agent协作：🤝 握手/协作图标
  - 信任安全：🛡️ 盾牌/安全图标
  - 工具调用：🔧 工具/齿轮图标

**色彩**：
- 条形图：渐变蓝到橙
- 图标：橙色高亮
- 文字：深灰

---

### 配图3：记忆架构分层图

**位置**：热点2章节

**功能**：解释分层记忆概念

**尺寸**：900 x 500 px

**设计要求**：
- **形式**：三层金字塔或分层结构图
- **内容**：
  ```
  ┌────────────────────────────┐
  │  永久记忆                   │
  │  （知识库、历史记录）         │
  ├────────────────────────────┤
  │  长期记忆                   │
  │  （项目经历、用户偏好）       │
  ├────────────────────────────┤
  │  短期记忆                   │
  │  （当前对话上下文）          │
  └────────────────────────────┘
  ```
- **视觉**：
  - 每一层用不同深度的蓝色
  - 层与层之间有数据流动的箭头
  - 右侧可添加示例数据图标

**色彩**：
- 永久记忆：深蓝 #0D47A1
- 长期记忆：中蓝 #1976D2
- 短期记忆：浅蓝 #42A5F5

### 配图4：多Agent协作示意图

## 三、视觉层次规划

### 3.1 文章结构可视化

```
[封面图]
  ↓
标题 + 引言
  ↓
[5大热点信息图]
  ↓
热点1：自主决策
  ↓
热点2：记忆系统
  [记忆架构图]
  ↓
热点3：多Agent协作
  [协作示意图]
  ↓
热点4：信任安全
（无需配图，可用图标装饰）
  ↓
热点5：工具调用
（无需配图，可用图标装饰）
  ↓
[三大趋势演进图]
  ↓
应用建议
（列表形式，无需配图）
  ↓
[CTA引导图]
  ↓
结尾 + 行动召唤
```

---

### 3.2 信息密度控制

**原则**：
- 每300-500字配一张图
- 避免连续大段文字
- 图文比例约 1:3

**实际配图位置**：
1. 封面图（第0字）
2. 5大热点图（约300字）
3. 记忆架构图（约800字）
4. 协作示意图（约1200字）
5. 三大趋势图（约1800字）
6. CTA图（约2200字）

## 四、设计规范

### 4.1 尺寸规范

| 图片类型 | 宽度 | 高度 | 用途 |
|---------|------|------|------|
| 封面图 | 900 | 500 | 文章开头 |
| 内容图 | 900 | 500-600 | 正文配图 |
| 信息图 | 900 | 600 | 数据展示 |
| CTA图 | 900 | 383 | 行动召唤 |

**单位**：像素（px）
**格式**：PNG或JPG
**文件大小**：每张<500KB

---

### 4.2 字体规范

**标题文字**：
- 字体：思源黑体 / 苹方
- 大小：48-72px
- 颜色：深灰 #263238

**正文文字**：
- 字体：思源黑体 / 苹方
- 大小：16-24px
- 颜色：深灰 #263238

**强调文字**：
- 颜色：橙色 #FF6D00
- 大小：与正文相同或略大

---

### 4.3 图标规范

**风格**：线性图标（outline）
**来源**：
- 免费图标库：Iconify、Feather Icons
- 自定义：保持风格一致

**使用原则**：
- 简洁明了
- 功能性强
- 风格统一
