## 🔍 详细步骤

### Step 1: 打开知网高级检索

访问网址：https://kns.cnki.net/kns8s/AdvSearch

**注意事项**：
- 如遇安全验证，完成滑块验证
- 建议使用Chrome或Edge浏览器

### Step 2: 选择"学术期刊"类别

1. 在页面顶部找到导航标签
2. 点击 **"学术期刊"** 标签
3. 等待页面刷新，显示检索表单

### Step 3: 勾选SCI来源

1. 在检索表单中找到 **"来源类别"** 区域
2. 取消勾选 **"全部期刊"**（如已勾选）
3. 勾选 **"SCI"** 复选框

**可选来源**：
- SCI（Science Citation Index）
- EI（Engineering Index）
-北大核心
- CSSCI（南大核心）

### Step 4: 输入检索关键词

**主关键词**：
- 在第一个 **"主题"** 输入框中输入：`农业智能体`

**同义词扩展**（可选）：
- 农业智能体 + 智慧农业
- 农业智能体 + 农业机器人
- 农业智能体 + 农业AI

**多关键词检索**（如需要）：
- 点击 **"+"** 按钮添加检索行
- 选择 **"OR"** 关系
- 输入第二个关键词

### Step 5: 执行检索

点击 **"检索"** 按钮，等待结果加载。

### Step 6: 按被引量排序

1. 在检索结果页找到 **排序** 选项
2. 点击 **"被引"** 排序按钮
3. 确保按 **被引量降序** 排列（被引最多的在前）

### Step 7: 切换显示设置

1. 找到每页显示数量选项（通常在底部）
2. 选择 **"50"** 条/页
3. 切换到 **"摘要视图"** 或 **"详细视图"**

### Step 8: 导出检索结果

#### 方法1：知网自带导出

1. 勾选需要的论文（或全选）
2. 点击 **"导出"** 按钮
3. 选择 **"Excel"** 格式
4. 保存到团队文件夹：`/Users/jeanne/20-29科研工作/AI 科研/07-引用管理/检索结果/`

#### 方法2：手动整理模板

如果知网导出功能受限，使用以下Excel模板整理：

| 序号 | 标题 | 作者 | 来源期刊 | 发表时间 | 被引次数 | 下载次数 | 摘要 |
|------|------|------|----------|----------|----------|----------|------|
| 1 | | | | | | | |
| 2 | | | | | | | |
| ... | | | | | | | |

**建议字段**：
- 序号：1-100（按被引量排序）
- 标题：论文完整标题
- 作者：所有作者，用分号分隔
- 来源期刊：期刊名称
- 发表时间：年份和期次
- 被引次数：被引用数量
- 下载次数：下载量
- 摘要：完整摘要内容
- DOI：数字对象标识符（如有）

如果需要更精确的检索，可以组合关键词：

- 农业智能体 AND 深度学习

如需限定发表时间：
- 在检索页面选择 **"发表时间"** 选项
- 设置时间范围，例如：2019-2026

## ⚠️ 常见问题

**Q1: 验证码一直出现怎么办？**

A: 知网有反爬机制，建议：
- 使用正常网络（避免VPN）
- 控制检索频率
- 手动完成验证码

**Q2: 检索结果为0？**

A: 尝试：
- 检查是否勾选了SCI
- 使用更宽泛的关键词
- 尝试其他来源类型（如EI、北大核心）

**Q3: 如何批量导出摘要？**

A: 知网通常限制批量导出，可以：
- 逐页复制粘贴
- 使用知网导出功能（可能有条数限制）
- 手动整理重要论文
